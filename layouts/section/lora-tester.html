FAQ

What is in the package?

You will get Core Module, Mini Battery Module, LoRa Module, LCD Module and Enclosure. If you order GPS version you will also get GPS Module, Split Module and bigger enclosure.

Is it one-case solution?

No. You can completely change behaviour of this kit. For example, it can measure temperature and light intensity and send it trough LoRa Network or you can extend it with one of many BigClown IoT Kit modules.

What is BigClown IoT Kit?

It is kit for students, teachers and makers. BigClown is a modular system that makes building IoT solutions simple, linking theory and practice. It comes with practical components for educators and learners to experiment and share specialist technical know-how without fuss. We believe IoT should be accessible, engaging and fun, so we can innovate and inspire the inventors of tomorrow, today, together.
